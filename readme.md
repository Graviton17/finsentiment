# FinSentiment Analyzer

FinSentiment Analyzer is a Streamlit app that uses FinBERT to analyze the sentiment of financial news articles. It supports fetching news from Finnhub and Yahoo Finance, and can also analyze text extracted from uploaded PDF files.

## Features

- Fetches recent news articles for a given stock ticker from Finnhub or Yahoo Finance.
- Analyzes the sentiment of each article using FinBERT.
- Displays results in an interactive dashboard with pie charts, bar charts, and timelines.
- Allows users to upload financial PDFs for sentiment analysis.
